Example of Live Leaflet Map

St. Louis Population Density (2017)

During class last night, we worked up to building a very similar map to the one served up below. The only difference between the maps we were working on and what is displayed here is that I’m using the Jenk’s natural breaks technique for determining the mapping classes.

How I Made This Map

If you want to re-create this map, you’ll need to download the neighborhood change shapefile. The code I used is below. The cp_bins() function is a custom one that I wrote to calculate the Jenks breaks. This is used to calculate the palette used on the map. Another departure from last night is that I stored the population density in a new variable, pop17_den, that I refer to throughout the leaflet() call. This makes the code a bit easier to read and reduces the chances of typos. We’ll talk about creating new variables in a couple of weeks!
